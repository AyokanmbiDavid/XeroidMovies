# Xeroid - Movie Discovery App

A lightweight, mobile-first web application for searching and discovering TV shows and movies.

## 🚀 Features
- **Live Search:** Powered by the TVMaze API.
- **Mobile-First Design:** Optimized for 3GB RAM devices and small screens.
- **Glassmorphism UI:** Clean, modern interface with skeleton loading effects.

## 🛠️ Built With
- HTML5 & Tailwind CSS (CDN)
- Vanilla JavaScript
- TVMaze API

## 📱 Mobile Development Note
This project was developed entirely on a mobile device. It uses the Tailwind CDN to avoid heavy build processes, making it ideal for low-RAM environments.
